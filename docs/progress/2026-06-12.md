## 4. Akşam planı (2 submission hakkı)

Config'ler hazır, `git pull` sonrası:

```bash
nohup bash -c './run.sh exp018 && ./run.sh exp019 && ./run.sh blend exp001 exp002 exp003 exp004 exp005 exp006 exp007 exp008 exp009 exp010 exp011 exp012 exp013 exp014 exp015 exp016 exp018 exp019' > logs/evening.log 2>&1 &
```

1. **exp018 — mDeBERTa-v3-base** (~50 dk GPU): farklı pre-training mimarisi (disentangled attention),
   BERT/XLM-R'dan farklı hata deseni → blend'e son text çeşitliliği şansı.
2. **exp019 — CatBoost @ kitchen-sink-v3 feature seti** (~1.5 saat HPO'lu): CatBoost şimdiye dek hep
   text-OOF'suz feature setinde koştu; en güçlü feature setinde algoritmik çeşitlilik hiç denenmedi.
3. Blend'i **exp017 HARİÇ** üye listesiyle koş (yukarıdaki komutta hazır). CV < 73.89 ise submit.

Beklenti dürüstçe: −0.1/−0.3 bandı; top-5 için ~−0.9 gerekiyor, garantisi yok. Pazar bitişten önce
en iyi public skorlu submission'ın seçili olduğundan emin ol (Kaggle final seçimi).
